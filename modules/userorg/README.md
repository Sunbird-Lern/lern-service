# UserOrg Service - Build & Deployment Guide

This module contains the UserOrg service. Below are the instructions for building the service and creating its Docker image.

## Prerequisites

- **Java 11**
- **Maven 3.6+**
- **Docker**

## Build & Deployment Scripts

The automation scripts are located in the `/scripts/userorg/` directory at the project root.

### 1. Build Service
Builds the service and generates the distribution artifact (`.zip`).

**Note:** Run these commands from the project root.

```bash
# Build without running tests (Default)
./scripts/userorg/build.sh

# Build and run tests
./scripts/userorg/build.sh -t
```

---

### 2. Build Docker Image
Creates a Docker image using the distribution artifact generated by the build script.

```bash
# Basic build (Image: userorg-service:latest)
./scripts/userorg/docker-build.sh

# Build with custom name and repo
./scripts/userorg/docker-build.sh -r my-registry -n userorg -t v1.0.0
```

**Parameters:**
- `-r, --repo`: Registry/Repository prefix (e.g., `sunbird`).
- `-n, --name`: Image name (default: `userorg-service`).
- `-t, --tag` : Image tag (default: `latest`).

---

## Complete Workflow Example

To build, containerize, and run the service:

1. **Build the application**:
   ```bash
   ./scripts/userorg/build.sh
   ```

2. **Build the Docker image**:
   ```bash
   ./scripts/userorg/docker-build.sh -r my-docker-hub-id
   ```

3. **Push the image**:
   ```bash
   docker push my-docker-hub-id/userorg-service:latest
   ```
